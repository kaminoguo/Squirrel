# Contributing to Squirrel

Thanks for your interest in contributing to Squirrel!

## Quick Start

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/Squirrel.git`
3. Create a branch: `git checkout -b yourname/feature-name`
4. Make your changes
5. Test your changes
6. Submit a Pull Request

## Development Setup

See [docs/DEVELOPMENT.md](docs/DEVELOPMENT.md) for detailed setup instructions.

## Before Submitting a PR

- [ ] Code follows project style
- [ ] Tests pass locally
- [ ] Linter passes
- [ ] Commits are in English and follow format: `type(scope): description`

## Pull Request Guidelines

- Create focused PRs (one feature/fix per PR)
- Write clear PR descriptions
- Link related issues
- Wait for 1 approval before merging

## Questions?

Feel free to open an issue for questions or discussion.
